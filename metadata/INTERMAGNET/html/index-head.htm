HAPI Server for INTERMAGNET data

This is a pass-though server of the magnetometer data maintained by intermagnet.org and stored in IAGA-2002-formatted files at ftp://ftp.seismo.nrcan.gc.ca/intermagnet/.

This server responds to requests by fetching the required files and concatenating or subsetting them (as needed). The numbers in a response will exactly match the numbers in the contributing files (each containing 1 day of data). The metadata for the files used to for the response can be found by either (1) making a request for metadata (see example below) or (2) reading the headers in the IAGA-2002-formatted files associated with the requested timerange.

The datasets are available at three quality levels: preliminary, quasi-definitive, and definitive and two time resolutions: 1-minute and 1-second. The following table lists all of the available datasets.

Because the coordinate system of data in a given dataset can change over the timerange of available data (typically 1-2 times), the parameters associated with each dataset are "Component 1-4" instead of, e.g., X, Y, Z, F. The coordinate system associated with data from a given time range can be determined using a request for metadata that is stored in the files associated with that time range. The reported coordinate system on the last available date of data is shown for each dataset in the table below in the Reported column.

For example:

  • List coordinate system of data from the Boulder, Colorado magnetometer station:
    http://hapi-server.org/INTERMAGNET/hapi/data?id=bou/definitive/minute/metadata&parameters=Reported&time.min=2018-12-28Z&time.max=2018-12-31Z.
  • List all metadata in the data files of data from the Boulder, Colorado magnetometer station:
    http://hapi-server.org/INTERMAGNET/hapi/data?id=bou/definitive/minute/metadata&time.min=2018-12-28Z&time.max=2018-12-31Z
